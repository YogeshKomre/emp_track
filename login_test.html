üîç Troubleshooting Steps

  1. Check Browser Console: Open Developer Tools (F12) and check for any JavaScript errors
  2. Verify Username: Make sure you're using the exact username as shown above (case-sensitive)
  3. Check Role Selection: Ensure you've selected the correct role for the user
  4. Clear Browser Cache: Try clearing your browser cache and cookies
  5. Check File Path: Make sure you're accessing the correct HTML file

Common Issues:

  • "Name not found" error: Username doesn't match exactly or wrong role selected
  • "Manager name not found" error: Manager name not in the list or wrong role
  • "SPM access is restricted" error: Username not in SPM team or wrong role
  • Page not loading: Check if all files (script.js, styles.css) are in the same folder
